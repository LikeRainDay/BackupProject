# 公共模块

> 此模块将项目中  多个模块公用的内容进行拆分到此模块中。其他模块使用公用内容只需集成对应模块即可。


## andy-common-bean

> 此模块作为公用的实体bean 以及一些公用的 异常 信息 和 controller的异常捕获。


## andy-common-dao
> 此模块作为公用的 数据访问层面的内容封装

## andy-common-entity

> 此模块作为公用的 Jpa实体的存放处

## andy-common-feign

> 此模块作为SpringCloud的RestTemplate的服务消费，方便提供其他模块快速使用其他服务暴露出来的接口。

## andy-common-urils

> 此模块作为 所用公用的工具类

## andy-swagger

> 此模块作为Swagger2 文档生成器使用
